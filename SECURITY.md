Reporting a Vulnerability
At SyncPointFlow, we take the security of the Vantage Vault seriously. If you discover a vulnerability, we appreciate your help in disclosing it to us responsibly.

🚩 How to Report
Please do not open a public GitHub issue for security vulnerabilities. Instead, use one of the following secure methods:

Discord Report: Use the !report command in our official Discord server. This will send a private alert to our lead developers.

Discord Ticket: Make a ticket in the Discord server in the support channel.

Private Message: Contact a Lead Developer (Vantage staff) directly on Discord.

🕒 What to Expect
Acknowledgement: You will receive a response within 24 hours confirming we have received the report.

Updates: We will provide status updates every 48 hours while the vulnerability is being investigated and patched.

Public Disclosure: Once a fix is deployed, we will credit the researcher (if desired) in our #announcements and GitHub Release notes.

🧪 Alpha Disclaimer
Because this is an Alpha build, we are focused on hardening the AES-256 implementation and Database Syncing. We prioritize reports involving:

Encryption bypasses.

Unauthorized database access.

Credential leakage.

Thank you for helping us secure the flow.
